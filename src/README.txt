Проект Sensor
---------------
Состояние цвета и веса вещества измеряется датчиками.
При температуре вещества 0 и ниже градусов, его вес 1 г,
а цвет - белый.
При повышении температуры, цвет изменяется на черный, а
вес меняется на 2 г за каждый градус.

Завершите проект, используя паттерн Наблюдатель.
Архив завершенного проекта загрузите в LMS.
